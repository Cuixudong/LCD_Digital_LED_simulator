# LCD_Digital_LED_simulator

基于LCD的数码管模拟器，可以在任何界面显示任意大小的数码管字体，支持有效及无效部分的空心及实心绘制，可以自定义数码管体宽度及尺寸坐标等

HAL_LIB压缩包需要解压后参与编译

演示效果：https://www.bilibili.com/video/BV1eb411M7Rv
